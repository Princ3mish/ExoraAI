# Integrations and Dependencies

## Core Integrations
- **PostgreSQL**: Primary data store, integrated via Prisma ORM (`DATABASE_URL` env variable required).

## Frontend Dependencies
- **Framer Motion**: For UI animations and transitions.
- **Lucide React**: For icon assets.
- **Tailwind Ecosystem**: Autoprefixer, PostCSS, tailwind-merge, clsx.

## Backend Dependencies
- **Winston**: Centralized logging system.
- **CORS**: Cross-origin resource sharing middleware enabled.
- **Express Global Error Handler**: Custom error management pattern used.
