# Technology Stack

## Frontend
- **Framework**: React 19.2 (w/ Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, PostCSS
- **UI Components**: Custom UI components (likely Shadcn UI derived given `clsx` and `tailwind-merge` in `lib/utils` and `components/ui`), Lucide React for icons
- **Animations**: Framer Motion
- **Tooling**: ESLint, TypeScript compiler, Vite

## Backend
- **Environment**: Node.js
- **Framework**: Express 5.2.1
- **Database ORM**: Prisma 5.22.0
- **Database Engine**: PostgreSQL
- **Language**: JavaScript (ES Modules standard)
- **Logging**: Winston
- **Tooling**: Nodemon (dev), ESLint
